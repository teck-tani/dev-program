HTML 색상표

RGB, HEX, HSL 색상 코드를 쉽게 찾고 복사하세요. 웹 디자인과 CSS 작업에 유용한 색상 정보를 제공합니다.

HEX: #4287f5

RGB: rgb(66, 135, 245)

HSL: hsl(217, 90%, 61%)

HTML 색상표 주요 기능

  • 다양한 형식의 색상 코드: HEX, RGB, HSL 등 모든 형식의 색상 코드 제공
  • 색상 선택기: 직관적인 색상 선택기로 원하는 색상을 쉽게 찾을 수 있습니다
  • 원클릭 복사: 색상 코드를 클릭 한 번으로 복사하여 바로 사용 가능
  • 색상 검색: 이름이나 코드로 색상을 검색할 수 있는 기능
  • 표준 색상 라이브러리: HTML과 CSS에서 사용되는 모든 표준 색상 제공
  • 커스텀 색상 추가: 자신만의 색상을 선택하고 저장 가능
  • 무료 사용: 모든 기능을 무료로 제공
  • 회원가입 필요 없음: 별도의 가입 절차 없이 즉시 사용 가능

HTML 색상표 사용 방법

1. 색상 찾기: 페이지에 표시된 색상 팔레트에서 원하는 색상을 찾거나, 색상 선택기를 사용하여 원하는 색상을 선택하세요.

2. 색상 정보 확인: 선택한 색상의 HEX, RGB, HSL 값이 자동으로 표시됩니다.

3. 색상 코드 복사: '복사' 버튼을 클릭하여 필요한 형식의 색상 코드를 클립보드에 복사하세요.

4. 웹 디자인에 활용: 복사한 색상 코드를 HTML, CSS 또는 기타 디자인 프로그램에 붙여넣어 사용하세요.

5. 색상 검색: 검색창을 사용하여 색상 이름이나 코드를 검색할 수 있습니다.

6. 색상 저장: '색상 추가' 버튼을 클릭하여 현재 선택한 색상을 팔레트에 추가할 수 있습니다.
