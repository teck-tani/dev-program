## 4. 개선 로드맵 — 5단계

### 핵심 전략: `jsbarcode` → `bwip-js` 라이브러리 마이그레이션

현재 `jsbarcode` (1D 전용, 17종)을 `bwip-js` (1D+2D, 100+ 종)로 교체하면:
- ✅ 2D 바코드 즉시 지원 (QR, DataMatrix, PDF417, Aztec)
- ✅ 색상 커스텀 내장 (`barcolor`, `backgroundcolor`, `textcolor`)
- ✅ 이미지 회전 내장 (`rotate` 파라미터)
- ✅ 폰트 설정 내장 (`textfont`, `textsize`)
- ✅ 여백 제어 내장 (`paddingwidth`, `paddingheight`)
- ✅ DPI 스케일링 내장 (`scale` 파라미터)

**한 번의 마이그레이션으로 6가지 갭을 동시 해결.**

---

### Tier 1: 핵심 기반 — 라이브러리 마이그레이션 + 2D 바코드 ★★★★★

#### 1.1 bwip-js 마이그레이션
- `jsbarcode` 제거, `bwip-js` 도입
- 렌더링 엔진 전면 교체 (SVG ref → bwip-js toSVG/toCanvas)
- 타입 식별자 매핑 (CODE128→code128, EAN-13→ean13 등)
- 검증 로직 단순화 (bwip-js 내부 검증 활용)
- **복잡도**: L | **수정 파일**: `BarcodeGenerator.tsx`, `package.json`

#### 1.2 2D 바코드 + 확장 포맷 (17종 → 40+종)
- **카테고리 UI**: 1D 선형 / 2D 매트릭스 / GS1 DataBar / 우편
- **추가 2D (5종)**: QR Code, Data Matrix, PDF417, Aztec, Micro QR
- **추가 1D (6종)**: CODE93, GS1-128, ISBN, ISSN, ISMN, UPC-E
- **추가 GS1 DataBar (4종)**: Omnidirectional, Limited, Expanded, Stacked
- **추가 우편 (6종)**: USPS POSTNET, Intelligent Mail, Royal Mail, Japan Post, AusPost, KIX
- **QR 전략**: 기본 QR 포함, "더 많은 QR 기능 →" 링크로 /qr-generator 연결
- **복잡도**: M | **수정 파일**: `BarcodeGenerator.tsx`, `ko.json`, `en.json`

#### 1.3 메타/스키마 버그 수정
- "16종" → 실제 포맷 수로 정정
- WebApplication featureList 전체 카테고리 반영
- **복잡도**: S | **수정 파일**: `page.tsx`, `ko.json`, `en.json`

---

### Tier 2: 시각적 커스터마이징 + 출력 강화 ★★★★

#### 2.1 색상 커스터마이징 (바/배경/텍스트)
- QR 생성기의 `type="color"` 패턴 재사용
- bwip-js: `barcolor`, `backgroundcolor`, `textcolor`
- **복잡도**: S (30분)

#### 2.2 이미지 회전 (0°/90°/180°/270°)
- bwip-js: `rotate` = `N`/`R`/`I`/`L`
- 4버튼 토글 UI
- **복잡도**: S (20분)

#### 2.3 인쇄 기능
- `window.print()` 호출 버튼 추가
- @media print CSS 이미 존재 → 레이아웃 옵션 (1/2/4열) 추가
- **복잡도**: S (20분)

#### 2.4 DPI/해상도 제어
- 72/150/300/600 DPI 선택
- PNG 다운로드 시 `scale` 배수 적용
- **복잡도**: S (30분)

#### 2.5 PDF 출력
- `pdf-lib` (이미 설치됨) 활용
- 단일 페이지 or 라벨 그리드 레이아웃
- **복잡도**: M (1-2시간)

---

### Tier 3: 고급 생성 기능 ★★★

#### 3.1 시퀀스/자동증가 생성기
- 접두사 + 시작번호 + 끝번호 + 접미사 + 제로패딩
- 예: `SKU-001` ~ `SKU-050` 자동 생성
- **복잡도**: M (1-2시간)

#### 3.2 여백(Quiet Zone) 제어
- bwip-js: `paddingwidth`, `paddingheight`

- **복잡도**: S (15분)

#### 3.3 폰트 커스터마이징
- 패밀리 (monospace/Arial/Helvetica/Times/Courier)
- 크기 (8-24pt), 볼드 토글
- **복잡도**: S (30분)

#### 3.4 CSV 파일 임포트 + 열 매핑
- `.csv`, `.tsv`, `.txt` 업로드
- 열 미리보기 + 선택 UI
- **복잡도**: M (1-2시간)

#### 3.5 임베드 코드 / 공유 링크

- **복잡도**: S (30분)

- **복잡도**: L (3-4시간) | **신규 파일**: `BarcodeScanner.tsx`

- 바코드 영역은 흰 배경 유지 (스캔 가능성)
- **복잡도**: M (1-2시간)
