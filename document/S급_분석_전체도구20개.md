# S급 달성을 위한 최근 추가 도구 20개 분석

> 분석일: 2026-02-13
> 대상: 1-2. 고수요 도구 추가 항목 전체 (금융 8 + 개발 8 + 유틸 4)

## 등급 기준

- **S급** = 해당 분야 최고 수준 (regex101, jwt.io, OmniCalculator급)
- 현재 대부분 **B~A급** (기본 기능 충실, 차별화 부족)

## 1. 금융/생활 계산기 (8개)

### 🔴 긴급 개선 (현재 B급)

| 도구 | 현재 등급 | 핵심 부족 기능 |
|------|----------|---------------|
| **프리랜서 3.3%** | B | 원천징수만 계산 → 종합소득세/경비율/실제 세금부담 없음. 너무 단순 |
| **주식 수익률** | B | 단일 거래만 가능 → 포트폴리오/평단가/DCA 계산 없음, 차트 없음 |

### 🟠 높은 우선순위 (현재 A-)

| 도구 | 현재 등급 | 핵심 부족 기능 |
|------|----------|---------------|
| **대출 상환** | A- | 상환 스케줄 차트 없음, "추가 상환 시 절감액" 계산 없음, PDF 내보내기 없음 |
| **BMI 계산기** | A- | 파운드/인치 단위 없음, 이상체중(Devine 공식) 없음, 체지방률 추정 없음 |
| **4대보험** | A- | 연간/5년 전망 없음, 급여 인상 시뮬레이션 없음, 파이차트 없음 |

### 🟢 양호 (현재 A급)

| 도구 | 현재 등급 | 소폭 개선 사항 |
|------|----------|---------------|
| **부가세 계산기** | A | 일괄 계산, 간이 세금계산서 출력 |
| **전월세 전환** | A | 지역별 시세 안내, 2년 총비용 비교 |
| **할인율/마진** | A | 중복 할인 계산 (30%+추가10%), 비교 테이블 |

### 금융 계산기 상세 부족 기능

#### BMI 계산기
- [ ] 파운드/인치(Imperial) 단위 토글
- [ ] 이상체중 계산 (Devine, Robinson, Miller 공식)
- [ ] 체지방률 추정 (Navy method)
- [ ] BMI 기록 추적 (localStorage)
- [ ] 결과 차트/시각화 개선

#### 부가세 계산기
- [ ] 일괄(다건) 계산 모드
- [ ] 간이 세금계산서 출력
- [ ] 히스토리에 날짜/라벨 추가

#### 대출 상환 계산기
- [ ] 상환 스케줄 차트 (원금 vs 이자 시각화)
- [ ] 추가 상환 시뮬레이션 ("매월 10만원 더 내면 N개월 단축")
- [ ] 대환대출 비교
- [ ] PDF/Excel 내보내기
- [ ] 30년(360행) 스케줄 테이블 페이지네이션

#### 전월세 전환 계산기
- [ ] 지역별 평균 전환율 안내 (서울/경기/지방)
- [ ] 2년 총비용 비교 (전세 vs 월세)
- [ ] 기회비용 계산 (전세금 투자 수익)

#### 4대보험 계산기
- [ ] 연간/5년 전망 표
- [ ] 급여 인상 시나리오 시뮬레이션
- [ ] 보험료 비율 파이차트
- [ ] 세금 공제 정보

#### 프리랜서 3.3% 계산기
- [ ] 종합소득세 추정 (경비율 적용)
- [ ] 분기별 세금 적립 안내
- [ ] 사업자등록 vs 프리랜서 비교
- [ ] 세금계산서 생성

#### 주식 수익률 계산기
- [ ] 포트폴리오 관리 (여러 종목 평단가)
- [ ] DCA(분할매수) 시뮬레이션
- [ ] 손익 차트
- [ ] 배당 수익률 계산
- [ ] 목표가 계산 ("X% 수익 위한 매도가")

#### 할인율/마진 계산기
- [ ] 중복 할인 계산 (30% + 추가 10%)
- [ ] 여러 시나리오 비교 테이블
- [ ] 손익분기점 분석

## 2. 개발자 도구 (8개)

### 🔴 긴급 (현재 B급)

| 도구 | 현재 등급 | 핵심 부족 기능 |
|------|----------|---------------|
| **JWT 디코더** | B | 서명 검증 불가 (jwt.io 핵심), 인코딩 모드 없음, 알고리즘 설명 없음 |

### 🟠 높은 우선순위 (현재 A-)

| 도구 | 현재 등급 | 핵심 부족 기능 |
|------|----------|---------------|
| **정규식 테스터** | A- | 패턴 해석 패널 없음 (regex101 핵심), URL 공유 없음, 치트시트 없음 |
| **UUID 생성기** | A- | v4만 지원 → v1/v7(타임스탬프) 없음, 유효성 검증 모드 없음 |
| **Markdown 미리보기** | A- | 체크리스트/이모지 미지원, PDF 내보내기 없음, 코드 구문 강조 없음 |

### 🟢 양호 (현재 A급)

| 도구 | 현재 등급 | 소폭 개선 사항 |
|------|----------|---------------|
| **해시 생성기** | A | 해시 검증 모드 (기대값 붙여넣기→일치 확인), HMAC 지원 |
| **타임스탬프** | A | 타임존 선택, 두 타임스탬프 간 차이 계산 |
| **HTML 엔티티** | A | 전체 엔티티 참조표 (현재 24개→2000+), 검색/필터 |
| **CSS Minifier** | A | 색상 축약 (#ffffff→#fff), 파일 다운로드 |

### 개발자 도구 상세 부족 기능

#### JWT 디코더
- [ ] 서명 검증 (Secret/Public Key 입력)
- [ ] JWT 인코딩(생성) 모드
- [ ] 알고리즘 설명 (HS256 vs RS256 vs ES256)
- [ ] 만료까지 남은 시간 카운트다운
- [ ] 표준 Claim 툴팁 (sub, aud, iss 등)
- [ ] "alg: none" 보안 경고

#### 정규식 테스터
- [ ] 패턴 해석 패널 (각 토큰이 무슨 의미인지 설명)
- [ ] URL 공유 (패턴+테스트 문자열 permalink)
- [ ] 정규식 치트시트 패널
- [ ] Named capture groups 시각화
- [ ] 실행 시간 표시
- [ ] 카타스트로픽 백트래킹 경고

#### UUID 생성기
- [ ] UUID v1 (타임스탬프 기반) 지원
- [ ] UUID v7 (시간 순서) 지원
- [ ] UUID 유효성 검증 모드
- [ ] UUID에서 타임스탬프 추출 (v1/v7)
- [ ] NIL UUID 버튼
- [ ] 파일(.txt) 다운로드

#### Markdown 미리보기
- [ ] GFM 체크리스트 `- [ ]` 지원
- [ ] 이모지 shortcode `:smile:` 지원
- [ ] 코드블록 구문 강조 (Prism.js/Highlight.js)
- [ ] PDF/HTML 내보내기
- [ ] 파일(.md) 업로드
- [ ] LaTeX 수식 지원
- [ ] 목차(TOC) 자동 생성

#### 해시 생성기
- [ ] 해시 검증 모드 (기대값 붙여넣기 → 일치/불일치 표시)
- [ ] HMAC 지원 (키 입력)
- [ ] 다중 파일 배치 해싱
- [ ] 대용량 파일 진행률 표시
- [ ] .sha256 체크섬 파일 생성

#### 타임스탬프 변환기
- [ ] 타임존 선택 (PST, EST, JST 등)
- [ ] 두 타임스탬프 간 차이 계산
- [ ] 배치 변환 (여러 타임스탬프 한번에)
- [ ] 프로그래밍 언어별 코드 스니펫
- [ ] "어제", "다음 월요일" 같은 프리셋

#### HTML 엔티티 변환기
- [ ] 전체 엔티티 참조표 확장 (24개 → 2000+)
- [ ] 검색/필터 기능
- [ ] Named vs Numeric 선택 옵션
- [ ] 유니코드 블록별 브라우징

#### CSS Minifier
- [ ] 고급 최적화 (색상 축약 #ffffff→#fff, 단축 속성)
- [ ] 최적화 상세 리포트 (무엇이 변경되었는지)
- [ ] .min.css 파일 다운로드
- [ ] CSS 유효성 검사/린트
- [ ] Before/After diff 비교

## 3. 전체 도구 공통 부족사항

| 항목 | 현황 | S급 기준 |
|------|------|---------|
| **차트/그래프** | 없음 (BMI 게이지만) | 대출 상환 그래프, 보험 파이차트, 주식 손익 차트 |
| **내보내기** | 복사만 가능 | PDF, Excel, CSV, 이미지 내보내기 |
| **결과 공유** | 없음 | URL 파라미터, 카카오톡/SNS 공유 |
| **키보드 단축키** | 없음 | Ctrl+Enter 계산, Ctrl+C 복사 등 |
| **인쇄 최적화** | 없음 | 인쇄용 레이아웃 (특히 금융 계산기) |

## 4. 추천 개선 순서 (임팩트 기준)

| 순위 | 도구 | 핵심 작업 | 예상 효과 |
|------|------|----------|----------|
| 1 | JWT 디코더 | 서명 검증 추가 | jwt.io와 경쟁 가능 |
| 2 | 대출 상환 | 차트 + 추가 상환 시뮬 | 금융 도구 핵심 차별화 |
| 3 | 정규식 테스터 | 해석 패널 + 치트시트 | regex101 대안 |
| 4 | 주식 수익률 | 포트폴리오 + 차트 | 투자자 유입 |
| 5 | UUID 생성기 | v1/v7 + 검증 모드 | 개발자 필수 기능 |
| 6 | Markdown | GFM + 구문 강조 | 개발자 체류시간 증가 |
