```
티스토리 블로그 포스팅을 작성해줘. 캔버스로 작성해줘.

## 주제
무료 온라인 이자 계산기 웹 도구 소개 (teck-tani.com/ko/interest-calculator)

## 도구 핵심 정보
- 이름: 이자 계산기 | 예금 적금 단리 복리 - 만기 수령액 계산
- URL: https://teck-tani.com/ko/interest-calculator
- 개발자가 직접 만든 무료 웹 도구 (광고 없음, 회원가입 없음)
- PC/모바일 반응형, 다크모드 지원

## 이 도구를 왜 만들었는지 (개발 배경)
- 은행에서 적금/예금 가입할 때 "실제로 얼마 받을 수 있지?" 계산이 번거로움
- 기존 계산기들은 단순 계산만 되거나, 세금 계산이 빠져있거나, 중도해지 시뮬레이션이 없음
- 단리 vs 복리 차이를 직관적으로 비교할 수 있는 도구가 필요했음
- 목표 금액을 정해놓고 "매달 얼마 넣어야 하지?" 역산 기능이 있으면 좋겠다고 생각
- 그래서 세금 계산 + 중도해지 시뮬레이션 + 목표 역산까지 한 번에 가능한 올인원 이자 계산기를 개발

## 주요 기능 (반드시 모두 언급)
1. **예금/적금 계산**: 목돈 예금(한 번에 맡기기) vs 월 적립 적금 선택 가능
2. **단리/복리 선택**: 단리와 월복리 이자 계산 비교
3. **세금 옵션 3가지**: 일반과세(15.4%), 세금우대(1.4%), 비과세(0%) 선택
4. **중도해지 시뮬레이션**: 해지 시점 입력 → 약정금리 50% 적용 → 예상 손실액/손실률 표시
5. **목표 역산 모드**: 원하는 최종 수령액 입력 → 필요한 예치금 또는 월 납입액 자동 계산
6. **성장 바 차트**: 원금과 이자 수익 비율을 시각적으로 보여주는 가로 바 차트
7. **월별 상세 내역 표**: 월차별 원금/이자/누적액을 표로 제공, 복사 기능 포함
8. **결과 복사 & 공유**: 계산 결과를 텍스트로 복사하거나 카카오톡 등으로 공유 가능
9. **숫자 애니메이션**: 계산 결과가 부드럽게 올라가는 애니메이션 효과

## 사용법 (단계별)
1. 계산 방식 선택 (예금 또는 적금)
2. 이자 방식 선택 (단리 또는 복리)
3. 세금 옵션 선택 (일반과세 / 세금우대 / 비과세)
4. 금액 입력 (예치금액 또는 월 납입액)
5. 기간(개월)과 연 이자율(%) 입력
6. "이자 계산하기" 버튼 클릭
7. 결과 확인: 원금합계, 세전이자, 이자소득세, 최종수령액

## 활용법 (실제 시나리오)
- 은행 예적금 가입 전 실제 수령액 미리 계산
- 단리 vs 복리 상품 어떤 게 유리한지 비교
- "3년 뒤 1,000만원 만들려면 매달 얼마 넣어야 하지?" → 목표 역산 모드 활용
- 급하게 돈 빼야 할 때 중도해지 손실이 얼마나 되는지 시뮬레이션
- 비과세 혜택 받을 수 있는 경우 세금 차이 비교
- 월별 이자 누적 추이를 표로 확인하고 엑셀에 붙여넣기

## 글 작성 가이드라인
- 어투: 친근하고 정보성 있는 ~요 체 (너무 딱딱하지 않게)
- 분량: 2,000~3,000자 내외
- 구성: 도입(왜 필요한지) → 기능 소개 → 사용법 → 활용 팁 → 마무리
- SEO 키워드 자연스럽게 포함: 이자 계산기, 적금 계산기, 예금 계산기, 단리 복리 차이, 복리 계산기, 적금 이자 계산법, 예금 이자 높은 은행, 세후 이자 계산, 중도해지 이자, 목표 금액 역산
- 제목 예시: "내 적금, 만기 때 실제로 얼마 받을까? 무료 이자 계산기 추천"
- 소제목(h2, h3)을 적절히 사용해서 가독성 높이기
- 마지막에 도구 링크 자연스럽게 삽입: https://teck-tani.com/ko/interest-calculator
- 스크린샷 삽입 위치를 [스크린샷: 메인 화면], [스크린샷: 결과 화면] 등으로 표시해줘
```

## 참고: 스크린샷 촬영 추천 포인트

블로그 포스팅 시 아래 화면들을 캡처하면 좋습니다:

1. **메인 입력 화면** - 예금/적금 토글, 단리/복리 선택, 세금 옵션 3종이 보이는 상태
2. **계산 결과 화면** - 원금합계/세전이자/이자소득세/최종수령액 + 성장 바 차트
3. **월별 상세 내역 표** - 펼쳐진 상태의 월별 표
4. **중도해지 시뮬레이션** - 중도해지 토글 ON + 결과 (황색 경고 카드)
5. **목표 역산 모드** - 목표 금액 입력 → 필요 납입액 결과 (하늘색 카드)
6. **모바일 화면** - 반응형 레이아웃
7. **다크모드 화면** - 다크모드 상태의 결과 화면
